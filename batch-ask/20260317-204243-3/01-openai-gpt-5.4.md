# 思路分析

要留住客户，核心不是“问得多”，而是“问得准”。

客户愿不愿意继续合作，通常取决于这几点：

1. **你是否真的懂落地页的本质**
   - 落地页不是企业官网介绍页。
   - 它的任务是：**让某一类人，在某一个场景下，完成某一个动作。**

2. **你是否能快速抓住影响转化的关键变量**
   真正决定初版质量的，通常不是几十个细节，而是这几个核心：
   - 卖什么
   - 卖给谁
   - 用户为什么会买
   - 用户为什么现在不买
   - 希望用户做什么动作
   - 有什么证据让用户相信
   - 流量从哪里来

3. **你是否让客户感到“被专业地理解了”**
   所以提问要有两个特点：
   - **少而准**
   - **每个问题都直接服务于后续页面结构与转化设计**

4. **你是否能把“调研”做成一种专业引导**
   不是把问题扔给客户，而是：
   - 帮客户理解为什么要回答这个问题
   - 尽量提供选项，降低客户思考成本
   - 用客户能回答的语言提问，不用太多术语

# 调研设计原则

这次调研的目标，不是做完整品牌咨询，而是为了让同事可以直接进入落地页初版制作。

因此问题必须覆盖这 6 个模块：

| 模块 | 必要性 | 作用 |
|---|---:|---|
| 产品/服务是什么 | 必须 | 决定页面主叙事 |
| 目标用户是谁 | 必须 | 决定文案角度与视觉风格 |
| 核心转化目标是什么 | 必须 | 决定页面 CTA 设计 |
| 用户为什么选择你 | 必须 | 决定卖点表达 |
| 信任证明有哪些 | 必须 | 决定成交支撑模块 |
| 流量来源/投放场景 | 必须 | 决定首屏与信息密度 |

其他很多问题可以暂时不问，比如：
- 公司成立几年
- 团队规模
- 品牌故事
- 喜欢什么颜色
- 竞争对手有多少家
- 是否有长期愿景

这些都不是初版落地页必须信息，或者可以后续补。

# 三、为什么这 8 个问题刚刚好

# 六、最优提问顺序建议

问题顺序也很关键，建议按这个逻辑走：

1. 先问推广对象是什么  
2. 再问页面目标是什么  
3. 再问目标客户是谁  
4. 再问为什么会买  
5. 再问为什么不买  
6. 再问拿什么让客户相信  
7. 再问流量从哪来  
8. 最后确认边界

这个顺序的好处是：
- 从“业务事实”切入，客户容易回答
- 再过渡到“转化逻辑”，显得专业
- 最后确认限制条件，避免一开始把对话带进琐碎细节

# 七、不建议问的问题

为了控制问题数量，以下内容第一轮不建议主动问：

## 1. “您喜欢什么风格/颜色？”
原因：
- 客户通常说不清
- 初版应该先由业务逻辑决定风格方向
- 真有品牌规范，客户会主动提

## 2. “竞品有哪些，请全部发我”
原因：
- 容易把客户带入大工作量
- 初版制作不一定必须
- 如果确实需要，可第二轮再补

## 3. “公司详细介绍/发展历程”
原因：
- 对落地页转化帮助有限
- 除非客户品牌背书特别强，否则不是第一优先级

## 4. “请提供完整产品资料包”
原因：
- 范围太大
- 客户容易拖延
- 应该改成只要“能支撑转化”的素材

# 八、给你的最终建议：怎么做到“客户惊艳”

真正让客户惊艳的，不是提问形式花哨，而是这三点：

## 1. 让客户觉得你不是在“收资料”，而是在“诊断转化”
所以你的提问必须围绕：
- 用户是谁
- 为什么买
- 为什么犹豫
- 为什么相信

## 2. 让客户觉得你很节省他的时间
所以问题要：
- 少
- 好答
- 有选项
- 有示例

## 3. 让客户觉得你已经在脑子里开始搭页面了
所以每个问题都暗示：
- 这关系到标题
- 这关系到 CTA
- 这关系到卖点
- 这关系到信任模块

这样客户会明显感觉到：  
**你不是一个“做页面的人”，而是一个“懂成交的人”。**
